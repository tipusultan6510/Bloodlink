Quick Setup (5 min, free)
BloodLink uses Google Firebase (free) so all your friends share the same live donor list in real time. You only do this once.
1Create a free Firebase project

Go to console.firebase.google.com → Sign in with Google → Click Add project → Name it bloodlink → Click through to finish.

2Create Realtime Database

In your project: Build (left menu) → Realtime Database → Create database → Choose any region → Select Start in test mode → Click Enable.

3Set database rules to public

In Firebase Console → Realtime Database → Rules tab → Replace all text with the line below → Click Publish:

{ "rules": { ".read": true, ".write": true } }
4Paste your Firebase config here

Gear icon → Project settings → scroll to Your apps → click </> → Register app → copy the values from firebaseConfig.
